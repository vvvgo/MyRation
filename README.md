# MyRation — персональный AI-ассистент по питанию

## Задача

Люди с диетическими ограничениями (заболевания ЖКТ, аллергии, набор/сброс веса) каждый день решают: **что съесть прямо сейчас** из того, что доступно. Нужен не трекер постфактум, а собеседник, который знает твой профиль, помнит что ты ел сегодня и поможет выбрать в моменте.

### Формат: ассистент-собеседник

```
👤 Хочу пойти на обед, думаю Караваевы или ВкусВилл.
   Хочется чего-то лёгкого и сладкого

🤖 В Караваевых тебе подойдёт сочник (320 ккал) — запечённый,
   состав чистый. Во ВкусВилле — скир «Печёное яблоко» (144 ккал, 14 г белка).
   Но ты набрал пока 600 ккал — может, сочник + куриный суп?

👤 А сочник мне точно можно?

🤖 [ищет состав] Мука, творог, яйцо, сахар, масло.
   Ничего из твоего стоп-листа. Можно.
```

```
👤 Съел на завтрак кашу овсяную из ВкусВилл 280 гр, банан и йогурт Teos

🤖 Записал: каша (295) + банан (100) + Teos (167) = 562 ккал, 29 г белка.
   До цели ~2240. Не забудь лекарство.
```

```
👤 Вечером хочу заказать роллы, можно мне?

🤖 Зависит от какие. Классические с рисом и рыбой — ок.
   Но избегай:
   — темпура (жареное в кляре)
   — с острым соусом или васаби
   — с сырым луком

   Филадельфия, Калифорния, ролл с лососем — можно.
   Хочешь, найду КБЖУ конкретного сета?

👤 Да, вот ссылка на сет из Яндекс.Еды [ссылка]

🤖 [парсит страницу]
   Сет «Филадельфия микс» 8 шт — 420 ккал, Б:18 Ж:16 У:48.
   Состав: рис, лосось, сливочный сыр, огурец, нори.
   Всё ок для тебя. Если хочешь наесться — бери два сета,
   это 840 ккал и закроешь вечер.
```

### Для кого

- Пациенты с заболеваниями ЖКТ, которым врач назначил диету и медикаменты
- Люди, набирающие вес после болезни или по другим причинам
- Люди, снижающие вес с конкретными ограничениями
- Все, кому нужен персональный контроль питания с учётом медицинского контекста

### Какая боль сейчас

- **Нет советника в моменте**: стоишь в очереди и не знаешь что выбрать из меню
- **КБЖУ неизвестны**: в столовых и кафе редко указывают калорийность, искать вручную долго
- **Конфликт целей**: хочется сладкого, но непонятно что из сладкого разрешено
- **Самообман**: кажется что ешь достаточно, а по факту — дефицит 1000 ккал 
- **Лекарства привязаны к еде**: забыл выпить вовремя — эффект лечения снижается
- **Потеря контекста**: обычный чат-бот не помнит что ты ел утром

## Что делает PoC на демо

1. **Диалог** — понимает свободную речь, обсуждает варианты, помогает выбрать
2. **Память** — помнит профиль (диагнозы, лекарства, стоп-лист) между сессиями
3. **Поиск КБЖУ** — сам ищет в интернете по названию блюда, по ссылке на продукт или по внутренней базе
4. **Анализ меню** — текст или фото → разметка «можно/осторожно/нельзя»
5. **Трекер дня** — накапливает КБЖУ, показывает остаток до цели
6. **Лекарства** — напоминает о привязке к еде
7. **Недельный отчёт** — калории, паттерны, динамика

## Что НЕ делает PoC

- Не ставит диагнозы и не назначает лечение
- Не распознаёт еду по фото тарелки (только меню/этикетки)
- Не заказывает еду, не интегрируется с доставкой
